Class: Player

ClubCrawler.Objects.Player(config)

A class for representing the Player in the game world.

Constructor

new Player(config)

Constructs the player, gives it a physics body, selects a random Discord avatar to use as the image.

(static) preload()

Preloads assets needed for player. Currently only looks to 'images/discord-avatars.png' and 'atlas/discord-avatars.json' for player image data. Picks a random avatar from the Phaser.Scene each time the game is run. See Phaser docs for more info about atlases.
